# Visa to the moon 🚀🌕

## Run the server

#### 1 - Install dependencies

-   `npm install`
    (Please, Enter on you terminal)

#### 2 - Run the server

-   `serve .`
    (Please, Enter on you terminal).

## Show html without server.

-   `open index.html`
    (Please, Enter on you terminal).

## How Run SASS engine.

-   `npx sass ./styles/index.scss ./build/styles/index.css`
